# xen-minecraft

A unique modded experience for `Minecraft 1.7.10`.

## Requirements

- Java 25

## Running

### Client
`mill --disable-ticker --interactive runs.client.runDefault`

### Server
`mill --disable-ticker --interactive runs.server.runDefault`

### Client & Server
`mill --disable-ticker --interactive runs.client.runDefault + runs.server.runDefault`
